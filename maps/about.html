About NYC Restaurants

NYC Restaurants is a web-based mashup that incorporates data compiled on NYC restaurants I have either eaten at or been recommended to eat at. The data on each restaurant is stored in an XML file that is then mashed together with Google Maps to provide a geographic representation of the restaurant along with corresponding information on that restaurant. The mashup was developed using the Google Maps API and I relied heavily on the Google Maps API Group and Mike Williams' Google Maps API Tutorial.

The mashup is a work in progress and continues to be updated regularly. I encourage the submission of other restaurants and any feedback you can provide. Please do so by using the submit page.
